Для получени псевдо глубины растеризуемого фрагмента следует пользовать встроенной в GLSL переменной gl_FragCoord (только для фрагментного шейдера). gl_FragCoord.z содержит значение псевдо глубины в диапазоне [0;1] (NB!, это значение необходимо перевести в [-1;1], как при выводе матрицы проекции).
